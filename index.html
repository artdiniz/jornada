Sobre a Jornada

Um programa educacional diferenciado, com foco em uma experiência completa de formação para os programadores Java que superem as expectativas do mercado. Um circuito de cursos cuidadosamente preparados e um TCC com vivência prática de desenvolvimento.

A Jornada Caelum Back-end prepara o aluno para as melhores vagas do mercado com foco em Java na Web. Começamos com as boas práticas de Web e HTML5, CSS3, JavaScript até Web Design Responsivo. Em seguida, uma visão aprofundada do Java, Orientação a Objetos, JSP e Spring MVC. Por fim, uma incursão nas tecnologias mais usadas no mundo Java, com o Hibernate, JPA, JSF, CDI e outros componentes do Java EE. Além de tecnologias específicas, focamos em boas práticas, testes automatizados e mais.

Para finalizar, o aluno desenvolve um projeto completo de TCC aplicando os conhecimentos aprendidos e as habilidades desenvolvidas durante a Jornada. Tudo com orientação de um professor da Caelum sempre revisando e dando dicas de melhorias. Formado, o aluno recebe o Diploma da Jornada Caelum, e está mais que preparado para os desafios do mercado de trabalho.

Sobre o TCC

O trabalho de conclusão é um projeto Java Web prático proposto pelo professor a ser desenvolvido pelo aluno em casa. O professor vai acompanhar constantemente a evolução do desenvolvimento, tirando dúvidas e dando feedback e dicas de melhorias.

O projeto pode usar quaisquer tecnologias e técnicas aprendidas nos cursos. Deve ser desenvolvidos no prazo de 3 semanas, quando será entregue para avaliação final do professor.

O TCC é parte essencial da experiência de aprendizado da Jornada Caelum. É um diferencial importante e projeta o aluno para o mercado de trabalho.
